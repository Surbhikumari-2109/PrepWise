#  "PrepWise" - An AI-Powered Learning Assessment Platform

<img width="1877" height="876" alt="PrepWise-home" src="https://github.com/user-attachments/assets/4fcb4e81-fa76-4f83-9bac-6ae80e996c19" />


PrepWise is full-stack web application designed to help students evaluate their knowledge, identify weak areas, and improve through personalized Ai feedback.

Unlike traditional assessment platforms that only display scores, PrepWise provides AI-generated insights to help learners understand their mistakes and focus on the concepts that need improvement.

---

## # Key Features

- Secure Authentication 
- Subject-Based Assessments
- Multiple Difficulty Levels (Easy, Medium, Hard)
- Timer-Based Tests
- AI-Powered Feedback using Google Gemini
- Performance Analytics & Accuracy Tracking
- Assessment History
- Leaderboard System
- User Profile Dashboard
- Fully Responsive Design

---
### Workflow

- User logs in using JWT Authentication.-
- User selects a subject and difficulty level.
- Frontend sends requests to the backend.
- Questions are fetched and displayed.
- Assessment results are stored in MongoDB Atlas.
- Incorrect responses are sent to Gemini AI.
- Gemini generates personalized feedback and learning recommendations.
- User can track performance through Dashboard, History, Profile, and Leaderboard

## # Why PrepWise?

Most learning platforms only show scores after an assessment.

PrepWise goes a step further by analyzing incorrect responses and providing AI-generated feedback that helps learners:

- Identify weak topics
- Understand mistakes
- Focus on revision areas
- Improve overall performance
